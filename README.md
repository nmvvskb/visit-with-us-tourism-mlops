# Visit with Us - Wellness Tourism Package Purchase Prediction (MLOps Project)
Project Overview

This project implements an end-to-end MLOps pipeline for predicting whether a customer is likely to purchase the newly introduced Wellness Tourism Package offered by Visit with Us.

The solution automates the complete machine learning lifecycle, including data registration, data preparation, model training, experimentation tracking, model registration, deployment, and CI/CD automation using GitHub Actions.

The objective is to help marketing and sales teams identify potential customers before outreach, improve campaign effectiveness, and support data-driven decision-making.
